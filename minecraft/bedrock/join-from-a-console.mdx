Unirte a tu servidor de Minecraft Bedrock desde una consola requiere un método especial, ya que las consolas no permiten la conexión directa por IP.\
La solución es configurar un DNS personalizado que te redirigirá a la lista de servidores.

### Accede a la Configuración de Red

El primer paso es cambiar la configuración de red en tu consola.

- En el menú principal de tu consola, navega hasta **"Configuración"**.
- Selecciona **"Red"** o **"Configuración de Internet"**.

### Cambia la Configuración DNS Primaria

Dentro de las opciones de red, cambia la configuración de DNS de **"Automático"** a **"Manual"** y utiliza la siguiente dirección de DNS primaria.

```yaml
DNS Primario: 104.238.130.180
```

En caso de que el DNS primario no funcione, puedes usar los DNS secundarios como alternativa:

```yaml
DNS Secundario: 8.8.8.8 o 1.1.1.1
```

### Ingresa la Dirección del Servidor

Al unirte al servidor de redirección, aparecerá una pantalla pidiéndote la IP y el puerto de tu servidor.\
Ingresa la dirección IP de tu servidor de XeroHost y el puerto correspondiente para conectarte.

```yaml
Dirección del Servidor: 213.21.245.232
Puerto: 19132
```

Una vez que hayas ingresado la información, haz clic en el botón para unirte y serás redirigido a tu servidor.
