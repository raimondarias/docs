### Accede a la Sección de Consola

1. Entra al panel de control de XeroHost: https://panel.xerohost.net
2. Selecciona el servidor de Minecraft que deseas configurar.
3. En el menú de la izquierda, dirígete a la sección **"Console"**.

   ![Console Pn](/images/console.png)

### Activar/Desactivar la Whitelist

Para activar y desactivar la whitelist, solo necesitas ejecutar el comando en la consola o dentro del servidor:

1. Para activar la whitelist

   ```bash
   whitelist on
   ```
2. Para desactivar la whitelist

   ```bash
   whitelist off
   ```

### Añadir Jugador a la Whitelist

Ahora que la whitelist está activada, los jugadores que no estén en la lista no podrán unirse. Para añadir a tus amigos, usa el siguiente comando:

1. En la consola, escribe lo siguiente, reemplazando **"nombre_del_jugador"** con el nombre de usuario de Minecraft que deseas añadir:

   ```bash
   whitelist add nombre_del_jugador
   ```

### Comandos Extras

- Para ver la lista de jugadores:

  ```bash
  whitelist list
  ```
- Para remover un jugador:

  ```bash
  whitelist remove nombre_del_jugador
  ```
