### Accede a la Sección de Versiones

1. Entra al panel de control de XeroHost: https://panel.xerohost.net/
2. Selecciona el servidor de Minecraft que deseas configurar.
3. En el menú de la izquierda, dirígete a la sección **"Versions"**.

![Vista del menú Versions en el panel de control](https://billing.xerohost.net/images/kb/25_scrnli_WRdFrEqErOcrfB.png)

### Elige el Tipo y la Versión

1. Selecciona el tipo de servidor que deseas instalar (por ejemplo, **Paper**). <img src="https://billing.xerohost.net/images/kb/26_scrnli_iqjMK3DKbOLqb2.png" alt="Selección del tipo de servidor (Paper)" />
2. A continuación, se mostrarán todas las versiones disponibles. Elige la que quieras haciendo clic en ella. <img src="https://billing.xerohost.net/images/kb/27_scrnli_bSqF5WI8aoN0ti.png" alt="Selección de la versión específica del servidor" />

### Instala la Nueva Versión

Una vez que elijas la versión, aparecerá una ventana de confirmación con dos opciones importantes:

<Warning>
  1. **Wipe Server Files** (Eliminar archivos del servidor): **Atención:** Si activas esta opción, se **eliminarán** todos los archivos de tu servidor actual. Si deseas conservar algo (como mapas o mods/plugins), asegúrate de hacer una copia de seguridad antes de continuar.
  2. **Accept EULA** (Aceptar EULA): Debes marcar esta casilla para confirmar que has leído y aceptas el **EULA (Acuerdo de Licencia de Usuario Final)** de Minecraft, que es un requisito para poder ejecutar el servidor.
</Warning>

![Ventana de confirmación de instalación con opciones Wipe Server Files y Accept EULA](https://billing.xerohost.net/images/kb/28_scrnli_4K1L3mMyIoo3IZ.png)

### Confirma la Instalación

1. Una vez que hayas seleccionado las opciones deseadas, haz clic en el botón **"Install"**.
2. El panel comenzará la instalación de la nueva versión. Este proceso puede tardar unos segundos o minutos.
3. Cuando termine, podrás iniciar tu servidor con la nueva versión.

### Inicia el Servidor

¡Y listo! Ahora, cuando arranques tu servidor, iniciará con la nueva versión que elegiste.

1. Dirígete a la pestaña **"Console"** y haz clic en **"Start"**.
2. Espera a que el servidor se inicie.
3. Para verificar qué versión estás usando, puedes ejecutar el comando `/ver` o `/version`.

<Warning>
  Cambiar la versión de tu servidor es un proceso que se realiza desde el panel de control. Es vital seguir las precauciones necesarias para evitar la corrupción de tus datos.

  🚨 **ADVERTENCIA CRÍTICA: Compatibilidad de Versiones**

  **Solo es seguro subir de versión (ej. de 1.16 a 1.19).** Bajar a una versión inferior (Downgrade) **NO es compatible** y causará que los archivos de tu mundo (chunks y datos de jugador) se corrompan, haciendo que el servidor sea inestable o inutilizable. Si debes bajar de versión, **debes eliminar el mundo actual.**
</Warning>
