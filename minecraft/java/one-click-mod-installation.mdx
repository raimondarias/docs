sidebarTitle: "Instalar Mods"
---

### Detener el Servidor

Antes de instalar cualquier mod, es crucial que detengas tu servidor para evitar la corrupción de archivos. En la página principal de tu panel, haz clic en el botón **"Stop"**.

![Botón para detener el servidor](https://billing.xerohost.net/images/kb/76_scrnli_o3i1tjKT6qRfkC.png)

### Navegar a la Sección de Mods

En el menú de la izquierda, encontrarás la sección **"Mods"**. Haz clic en ella para acceder al gestor de instalación.

![Menú de mods en el panel de control](https://billing.xerohost.net/images/kb/89_scrnli_CkRAKU0A46gueO.png)

### Usar los Filtros para Buscar el Mod

Para encontrar el mod correcto, utiliza el buscador y los filtros. Es crucial que uses el filtro de **"Software"** para seleccionar el **"mod loader"** (por ejemplo, **Fabric**, **Forge** o **NeoForge**) que utiliza tu servidor. Luego, usa el filtro de **Providers** para elegir la plataforma de descarga (como **Modrinth** o **CurseForge**).

![Filtros de búsqueda para mods](https://billing.xerohost.net/images/kb/90_scrnli_W0EUps9LE6J4dx.png)

### Elegir y Instalar la Versión Correcta

Una vez que encuentres el mod, haz clic en el botón **"Install"**. Si necesitas una versión específica, haz clic en **"Versions"** para abrir la lista de versiones disponibles. En esta ventana, puedes usar los filtros de **Versions** y **Software** para asegurar la compatibilidad y luego hacer clic en el botón **"Install"**.

![Ventana de versiones disponibles para un mod](https://billing.xerohost.net/images/kb/91_scrnli_UFLeq3VGo6l76P.png)

### Iniciar el Servidor

Una vez que la instalación haya finalizado, puedes iniciar tu servidor nuevamente haciendo clic en el botón **"Start"** en la página principal. El nuevo mod se cargará automáticamente.
