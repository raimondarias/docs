### Detener el Servidor

Antes de añadir o modificar cualquier archivo del mundo, es fundamental que detengas el servidor para evitar que los archivos se corrompan.
En la página principal de tu panel, haz clic en el botón **Stop**.

<Frame>
  ![Stop](/images/games/stop.png)
</Frame>

### Navegar al Gestor de Archivos

En el menú de la izquierda, navega hasta la sección **Files** para acceder al gestor de archivos de tu servidor.

<Frame>
  ![Menú del gestor de archivos](/images/games/files.png)
</Frame>

### Localizar la Carpeta del Mundo

Una vez en el gestor de archivos, navega a la carpeta principal de tu mundo, que por lo general se llama `world`.\
Dentro de esta carpeta, encontrarás una subcarpeta llamada `datapacks`:

```bash
/home/container/
    - world/
        - datapacks/
    - logs/
    - server.jar
    - etc...
```

![Ubicación de la carpeta datapacks](https://billing.xerohost.net/images/kb/93_scrnli_kMZa3V82r72EOh.png)

### Subir el Datapack

Sube el archivo `.zip` del datapack directamente a la carpeta `datapacks`.\
**No descomprimas** el archivo `.zip`. Simplemente arrástralo y suéltalo o utiliza la función de carga del gestor de archivos.

![Subir archivo del datapack](https://billing.xerohost.net/images/kb/96_scrnli_G9ij10fpl7Av49.png)

### Iniciar el Servidor y Verificar

Una vez que el archivo haya sido subido, puedes iniciar el servidor nuevamente.\
El datapack se activará automáticamente. Puedes verificarlo con el siguiente comando en la consola:

```bash
datapack list
```

![datapack list](https://billing.xerohost.net/images/kb/97_scrnli_pGza0x5867CxZM.png)

Si el datapack aparece en la lista, se ha instalado correctamente.
