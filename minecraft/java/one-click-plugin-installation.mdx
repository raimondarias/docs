---

### Detener el Servidor

Antes de instalar cualquier plugin, es crucial que detengas tu servidor para evitar la corrupción de archivos. En la página principal de tu panel, haz clic en el botón **"Stop"**.

![Botón para detener el servidor](https://billing.xerohost.net/images/kb/76_scrnli_o3i1tjKT6qRfkC.png)

### Navegar a la Sección de Plugins

En el menú de la izquierda, encontrarás la sección **"Plugins"**. Haz clic en ella para acceder al gestor de instalación.

![Menú de plugins en el panel de control](https://billing.xerohost.net/images/kb/84_scrnli_o3Y4u2A6wXo5uK.png)

### Usar los Filtros para Buscar el Plugin

Para encontrar el plugin correcto, utiliza el buscador y los filtros. Es recomendable usar el filtro de **"Providers"** para seleccionar una fuente de confianza como **"SpigotMC"** o **"Modrinth"**.

![Filtros de búsqueda para plugins](https://billing.xerohost.net/images/kb/81_GmWdquW.png)

### Elegir y Instalar la Versión Correcta

Una vez que encuentres el plugin, tienes dos opciones:

- Haz clic en el botón **"Install"** para instalar la versión más reciente por defecto.
- Si necesitas una versión específica, haz clic en el botón **"Versions"** para abrir la lista de versiones disponibles.

En la ventana de versiones, usa el filtro de **"Software"** para asegurarte de que la versión del plugin sea compatible con tu software de servidor (como **PaperMC** o **Spigot**), y luego haz clic en **"Install"**.

![Ventana de versiones disponibles para un plugin](https://ejemplo.com/imagenes/ventana-versiones-plugins.png)

### Iniciar el Servidor

Una vez que la instalación haya finalizado, puedes iniciar tu servidor nuevamente haciendo clic en el botón **"Start"** en la página principal. El nuevo plugin se cargará automáticamente.
